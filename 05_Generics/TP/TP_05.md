## 📝 Deskripsi Kode
Dalam penyelesaian modul ini, saya mengembangkan sebuah fungsi logika menggunakan JavaScript yang dirancang untuk mengelola perhitungan statistik teks secara efisien. Struktur kode ini berpusat pada satu fungsi generik bernama hitung() yang mampu menangani dua parameter operasional berbeda, sehingga meminimalisir redundansi kode dan meningkatkan reabilitas program.

Saya mengimplementasikan logika kontrol aliran yang fleksibel untuk memproses data string berdasarkan input pengguna. Melalui parameter "semua", fungsi secara komprehensif menghitung seluruh unit karakter termasuk spasi. Sementara itu, mode "huruf" mengintegrasikan perintah continue untuk mengabaikan karakter spasi secara spesifik, sehingga aplikasi dapat memberikan hasil perhitungan jumlah huruf murni secara akurat sesuai dengan prinsip Don't Repeat Yourself (DRY).
